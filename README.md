# rfnsyhmi's tracker!

A full deployable React + Vite financial tracker for:
- Personal
- School-Brigade
- Work-Business

## Features
- Add, edit, and delete transactions
- Local storage persistence
- Automatic balance tracking
- Dashboard with totals and charts
- CSV export by section
- Reimbursement tracking for School-Brigade
- Netlify-ready deployment

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Netlify
- Push this project to GitHub
- Import the repo into Netlify
- Build command: `npm run build`
- Publish directory: `dist`
